 ## Part Two : CSS üíÉ
 So now we have a webpage, but let's be honest, it's pretty ugly! So let's make it pretty, with CSS! 
 
 - On the left, click on the item that reads ```style.css```, so we can start writing some css. Your CSS document should be blank, unline the HTML document. 
 
 - Write the code ```body{text-align:center;}``` in the CSS file. 

What this does is center everything that is in the ```<body>``` element. The ```body``` before the curly brackets is called a selector and what the CSS is applied to. Let's take a look!

<img width="1427" alt="Screen Shot 2021-03-25 at 1 06 33 AM" src="https://user-images.githubusercontent.com/51331370/112426675-59bfe300-8d06-11eb-98a7-0360cfec1bf9.png">


- Okay, now copy paste the following code into the top of your CSS document, above the ```body``` from before : ```@import url('https://fonts.googleapis.com/css2?family=Montserrat:wght@400;600&display=swap');``` This is going to import a certain font from a Font API : [Google Fonts](https://fonts.google.com/). Next we need to write the following code **inside** our body selector: ```font-family: 'Montserrat', sans-serif;``` 

The entirety of our CSS should look something like this: 

<img width="442" alt="Screen Shot 2021-03-25 at 1 18 13 AM" src="https://user-images.githubusercontent.com/51331370/112427675-fa62d280-8d07-11eb-8e82-74db0d1086be.png">

If you refresh your webpage again, this is what it should look like: 

<img width="1424" alt="Screen Shot 2021-03-25 at 1 11 54 AM" src="https://user-images.githubusercontent.com/51331370/112427137-19ad3000-8d07-11eb-9bc9-05ced881d71a.png">

It's coming together, but our paragraph text is a bit wide. 

- Next add the following code beneath the ```body``` selector in your CSS : ```p{ margin-left: 7%; margin-right: 7%;}``` This creates margin for the p element on both the right and left side that is 7% of the screen width.
